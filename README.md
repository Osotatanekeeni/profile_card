# Profile Card

A simple and elegant profile card showcasing personal details, including name, email, location, job title, and social media links. The card also displays the current UTC time dynamically.

## Table of Contents

- [Features](#features)
- [Installation](#installation)
- [Usage](#usage)
- [Technologies Used](#technologies-used)


## Features

- Displays personal details: name, email, location, and job title.
- Shows the current UTC time dynamically.
- Includes social media links with icons for GitHub, LinkedIn, and Twitter.
- Responsive design for different screen sizes.

## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/yourusername/profile_card.git
    ```

2. Navigate to the project directory:

    ```bash
    cd profile_card
    ```

3. Open [index.html](http://_vscodecontentref_/1) in your preferred web browser.

## Usage

1. Update the personal details in the [index.html](http://_vscodecontentref_/2) file.
2. Replace the social media links with your own URLs.
3. Customize the styles in the [style.css](http://_vscodecontentref_/3) file as needed.
4. Ensure the icons for social media links are placed in the [icons](http://_vscodecontentref_/4) directory.
